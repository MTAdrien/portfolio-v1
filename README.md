# Portfolio Développeur V1

Portfolio développeur moderne réalisé en HTML, CSS et JavaScript Vanilla, dans le cadre de ma formation Concepteur d'application/AI Software Engineer .

## Aperçu du projet

Ce projet à pour objectifs:

- Présenter mes compétences
- Montrer mes projets
- Construire une image professionnelle
- Préparer une alternance IA/Data
- Développer une présence web crédible

## Technologies utilisées

- HTML5
- CSS3
- JavaScript Vanilla

## Fonctionnalités

- Responsive design
- Animations légères
- Scroll smooth
- Navbar fixe
- Cartes projets modernes
- Bouton retour haut

## Vision

Projet orienté Full Stack, SaaS et IA appliquée.
